# PyroBot

## Overview

PyroBot is a Discord bot designed to help you manage your Pyrodactyl game servers directly from Discord. With simple commands, you can start, stop, restart servers, monitor their status, and perform basic administrative tasks without needing to access the Pyrodactyl panel.

## Features

- **Server Control**: Start, stop, and restart your game servers
- **Status Monitoring**: Check server status, resource usage, and uptime
- **Console Access**: Send commands and view console output
- **User Management**: Manage server permissions and user access
- **Notifications**: Receive alerts for server events and status changes

## Getting Started

PyroBot integrates with the Pyrodactyl Panel API to provide seamless server management through Discord's familiar interface.
