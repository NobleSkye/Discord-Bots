# Skye's Discord Bots

These are a collection of Discord bots created and maintained by Skye for various purposes.

## Bots

- **PyroBot**: A Discord bot for managing Pyrodactyl game servers.
- **PaggerBot**: A Discord bot for managing pagers and notifications.

## Installation
Download the repo/just the bot you want, install and follow the setup instructions in each bot's README file. usually with docker/docker-compose.
